# SYNC OUT GENERATOR

Generation of SYNC OUT

## Spec

* 5V pulse use PIN7
* 1.Standalone Sync Generator
    *2PPQ, 24PPQ, 48PPQ, 60-250BPM
* 2.MIDI-USB to Sync
    *2PPQ, 24PPQ

## Required

* compatible promicro (atmega32u4 5V 16MHz)
* Tiny4kOLED 2.2.2 or later
* MIDIUSB 1.0.5 or later
* MsTimer2 1.1 or later
